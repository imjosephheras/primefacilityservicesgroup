# PFS Calculator Suite

Professional calculator suite for Prime Facility Services Group.

## 🚀 Quick Setup

### For Hostinger:

1. **Rename the main folder** to something short like `calc` or `pfs`
2. **Upload** the entire folder to your `public_html`
3. **Access** at: `https://yourdomain.com/calc/`

### URLs Structure:
```
https://yourdomain.com/calc/             # Main login page
https://yourdomain.com/calc/kitchen/     # Kitchen calculator
https://yourdomain.com/calc/vent-hood/   # Vent Hood calculator
https://yourdomain.com/calc/timesheet/   # Timesheet calculator
https://yourdomain.com/calc/work-order/  # Work Order generator
```

## 📁 Folder Structure:
```
calc/
├── index.html      # Main page with login
├── kitchen/        # Kitchen cleaning calculator
├── vent-hood/      # Vent Hood cleaning calculator
├── timesheet/      # Timesheet calculator
├── work-order/     # Work Order generator
├── shared/         # Shared utilities and modules
└── .htaccess       # Server configuration
```

## 🔧 Features:
- **Kitchen Calculator**: Estimate cleaning time and costs for commercial kitchens
- **Vent Hood Calculator**: Calculate vent hood cleaning services with commission tracking
- **Timesheet Calculator**: Track employee hours and generate staffing proposals
- **Work Order Generator**: Create professional work orders for services

## ⚠️ Important:
- All folder names are lowercase without spaces
- The .htaccess file ensures proper routing
- Allow popups for PDF generation

## 🛠 Troubleshooting:

If navigation doesn't work:
1. Check that ALL files were uploaded
2. Verify folder names are exactly as shown above
3. Clear browser cache

For PDF issues:
1. Allow popups in browser
2. Use HTTPS not HTTP
3. Check browser console for errors
