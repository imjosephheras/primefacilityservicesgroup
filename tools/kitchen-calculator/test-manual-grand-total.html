Manual Grand Total Testing - Kitchen Calculator

Test 1: Basic Manual Grand Total Edit
  1. Open kitchen/index.html
  2. Set basic parameters:
    • Workers: 2
    • Hours: 8
    • Days: 1
  3. Note the calculated Grand Total
  4. Click the edit button next to Grand Total
  5. Enter a different value (e.g., if it shows $2,500, enter 3000)
  6. Press Enter or click outside the input
Expected Result:
  • The Grand Total should update to exactly $3,000
  • The Markup percentage should automatically adjust
  • A "Manual Adjustment" indicator should appear
  • The Markup row should briefly highlight
Test 2: Manual Edit with Rounding Enabled
  1. Ensure "Round to nearest $50" is checked in Pricing Options
  2. Set parameters to get a non-rounded total
  3. Edit the Grand Total to a specific value like $4,250
Expected Result:
  • The system should calculate the correct markup to achieve exactly $4,250
  • The total should remain at $4,250 (not rounded further)
Test 3: Manual Edit with Insurance and Fees
  1. Enable "Include Insurance" in the options
  2. Enable "Initial Service Fee" and set it to $200
  3. Calculate a quote
  4. Edit the Grand Total to a specific amount (e.g., $5,000)
Expected Result:
  • The system should correctly account for insurance and fees
  • The final Grand Total should be exactly $5,000
  • All components (base price, insurance, fees) should adjust proportionally
Test 4: Edge Cases
  1. Try entering a very low Grand Total (e.g., $100)
  2. Try entering a very high Grand Total (e.g., $50,000)
  3. Try entering the exact same value as currently shown
  4. Try entering 0 or negative values
Expected Result:
  • Very low values: Should set markup to 0% if necessary
  • Very high values: Should calculate high markup percentage
  • Same value: Should maintain current state
  • Invalid values: Should be rejected or ignored
Test 5: Interaction with Other Features
  1. Set a manual Grand Total
  2. Change other parameters (workers, hours, etc.)
  3. Observe what happens to the Grand Total
Expected Result:
  • Changing other parameters should recalculate with the custom markup
  • The Grand Total will change based on the new costs
  • The custom markup percentage should remain the same
Note: If you want to maintain a specific Grand Total while changing parameters, you'll need to re-edit the Grand Total after making changes.

Technical Details

The manual Grand Total edit works by:

  1. Calculating the required markup percentage to achieve the target Grand Total
  2. Accounting for all factors: insurance, franchise tax, initial fees, rounding
  3. Using an iterative approach when rounding is enabled to find the exact value
  4. Performing a verification step after calculation to ensure accuracy
  5. Fine-tuning the markup if the result doesn't match the target
